
This sample is a simple VB.NET desktop application that adds receive payments to QuickBooks using qbXML 
built with the MSXML6 DOM parser and QBFC.
Running the sample
------------------
Before running ReceivePayment.exe, make sure that .NET runtime is installed on the machine,
and QuickBooks is running with a company opened.
Building the sample
------------------
Please install latest QBSDK.
Open ReceivePayment.sln in Microsoft Visual Studio .NET and build the solution.
