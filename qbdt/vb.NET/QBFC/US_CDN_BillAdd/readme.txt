This sample is a simple VB.NET desktop application that is used to add the bill by using QBFC2 COM dll. 

Running the sample
------------------

Before running US_CDN_BillAdd.exe, make sure that .NET runtime is installed on the machine, 
and QuickBooks is running with a company opened. 

Building the sample
------------------
Please install latest QBSDK.
Open US_CDN_BillAdd.sln in Microsoft Visual Studio .NET and build the solution.
QBFC2Assembly.dll was generated with 'tlbimp' .NET Type Library to Assembly Convertor.
