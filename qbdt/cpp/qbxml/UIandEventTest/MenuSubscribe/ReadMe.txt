
This sample is a simple C++ desktop application that Adds UI Extension (menu) subscriptions to QuickBooks.
built with the MSXML6 DOM parser.
Running the sample
------------------
Before running MenuSubscribe.exe, make sure that QuickBooks is running with a company opened.
Building the sample
------------------
Please install latest QBSDK.
Open MenuSubscribe.sln in Microsoft Visual Studio and build the solution.
