========================================================================
    APPLICATION : hostquery Project Overview
========================================================================
This VC++2005 sample demonstrates the minimal qbXML required to perform 
a hostquery to QuickBooks. The user can view the qbXML request and 
response in messagebox. The intent of this sample is to demonstrate how 
to utilize QBXMLRP2 library using VC+2005.

This readme file contains a summary of what you will find in each of the files that
make up your hostquery application.

hostquery.vcproj
    This is the main project file for VC++ projects. It contains information about 
    the version of Visual C++ that was used, and information about the platforms, 
    configurations, and project features selected with the Application Wizard.

hostquery.cpp
    This is the main application source file.
    Contains the code to display the hostquery form.

Form1.h
    Contains the implementation of your hostquery form class and InitializeComponent() 
    function. It also contains the implementation when the button is clicked. 

AssemblyInfo.cpp
    Contains custom attributes for modifying assembly metadata.

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named hostquery.pch and a precompiled types file named StdAfx.obj.
